# Support / 支持说明

SiaoCut is under active development. There is no service-level agreement, guaranteed response time, live setup support, or production support for source builds and unsigned candidates.

SiaoCut 仍在开发中。源码构建与未签名候选包不提供服务等级协议、固定响应时间、实时安装协助或生产支持。

## Public issues / 公开问题

Use [GitHub Issues](https://github.com/ShawnSiao/siao-cut/issues) for reproducible bugs and bounded feature requests. Before submitting:

- confirm the issue on the latest relevant commit;
- include Windows version, SiaoCut commit, operation, expected result, and actual result;
- replace local paths with placeholders;
- remove media, transcripts, screenshots containing content, account names, tokens, and personal information; and
- state whether the build is from source, an unsigned candidate, a prerelease, or a formal Release.

可复现缺陷和范围明确的功能建议使用 [GitHub Issues](https://github.com/ShawnSiao/siao-cut/issues)。提交前需要：

- 在相关的最新提交上确认问题；
- 提供 Windows 版本、SiaoCut 提交、操作步骤、预期结果和实际结果；
- 使用占位符替换本机路径；
- 移除媒体、文稿、包含内容的截图、账号、令牌和个人信息；
- 说明运行的是源码、未签名候选包、prerelease 还是正式 Release。

## Out of scope / 不提供的支持

- Installation or debugging of third-party GPU drivers, CUDA, Python, SGLang, vLLM, MOSS, FFmpeg, or whisper.cpp beyond documented SiaoCut integration checks.
- Recovery of lost signing keys, deleted source media, or manually modified SQLite databases.
- Copyright, download authorization, publishing, or platform-account decisions.
- Review of private media or transcripts through a public issue.

- 第三方 GPU 驱动、CUDA、Python、SGLang、vLLM、MOSS、FFmpeg 或 whisper.cpp 的安装与调试；SiaoCut 文档中的集成检查除外。
- 丢失签名密钥、删除原片或手工修改 SQLite 后的数据恢复。
- 版权、下载授权、发布或平台账号决策。
- 通过公开 Issue 审阅私有媒体或文稿。

Security-sensitive reports follow [SECURITY.md](SECURITY.md). Creator Beta evidence follows [docs/english-creator-beta-feedback-template.md](docs/english-creator-beta-feedback-template.md) and remains outside the repository.

安全敏感问题按 [SECURITY.md](SECURITY.md) 处理。Creator Beta 证据使用 [docs/english-creator-beta-feedback-template.md](docs/english-creator-beta-feedback-template.md)，并保存在仓库外。
